Adiciona um ícone do WhatsApp no rodapé do site.

== Descrição ==

O WhatsApp AGY adiciona um ícone do WhatsApp no rodapé do seu site, permitindo que os visitantes entrem em contato com você diretamente pelo WhatsApp. O número do WhatsApp pode ser facilmente configurado através do painel administrativo do WordPress.

== Instalação ==

1. Faça o upload dos arquivos do plugin para o diretório `/wp-content/plugins/whatsapp-agy` ou instale o plugin diretamente através da tela de plugins do WordPress.
2. Ative o plugin através da tela 'Plugins' no WordPress.
3. Vá para 'WhatsApp AGY' no menu do painel administrativo e configure o número do WhatsApp.

== Perguntas Frequentes ==

= Como eu configuro o número do WhatsApp? =
Após ativar o plugin, vá para 'WhatsApp AGY' no menu do painel administrativo e insira o número do WhatsApp no formato internacional (código do país seguido pelo número).

= O ícone do WhatsApp não aparece no rodapé, o que eu faço? =
Certifique-se de que você tenha inserido um número de WhatsApp válido no formato internacional. Verifique também se o seu tema suporta hooks no rodapé.
